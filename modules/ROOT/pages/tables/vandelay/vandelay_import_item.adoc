=== Trigger Bodies

==== inherit_import_item_imported_as_fkey

*Function:* xref:functions/evergreen.adoc#vandelay_import_item_imported_as_inh_fkey[`evergreen.vandelay_import_item_imported_as_inh_fkey()`] +
*Timing:* AFTER INSERT OR UPDATE ROW

[source,plpgsql]
----

BEGIN
        IF NEW.imported_as IS NULL THEN
                RETURN NEW;
        END IF;
        PERFORM 1 FROM asset.copy WHERE id = NEW.imported_as;
        IF NOT FOUND THEN
                RAISE foreign_key_violation USING MESSAGE = FORMAT(
                        $$Referenced asset.copy id not found, imported_as:%s$$, NEW.imported_as
                );
        END IF;
        RETURN NEW;
END;
----
