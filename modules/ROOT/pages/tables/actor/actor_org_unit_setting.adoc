= actor.org_unit_setting
:description: Table reference for actor.org_unit_setting

xref:schemas/actor.adoc[↑ actor schema]


Org Unit settings

This table contains any arbitrary settings that a client
program would like to save for an org unit.


[CAUTION]
====
*Data-Modifying Triggers:* This table has BEFORE ROW trigger(s) that modify row data before write. Values you INSERT or UPDATE may differ from what is actually stored. See the <<triggers,Triggers>> section below.
====

[WARNING]
====
*Cascading Deletes:* Deleting rows from this table will cascade to: xref:tables/actor/actor_org_unit.adoc[actor.org_unit].
====

[WARNING]
====
*Deferrable Constraints:* The following FK constraints are deferrable — they are checked at transaction end, not statement end: `org_unit_setting_name_fkey`, `org_unit_setting_org_unit_fkey`.
====

[NOTE]
====
*Trigger Side Effects:* Writing to this table automatically triggers writes to other tables:

* `log_ous_change` → INSERT xref:tables/config/config_org_unit_setting_type_log.adoc[config.org_unit_setting_type_log]
====

=== Trigger Bodies

==== log_ous_change

*Function:* xref:functions/evergreen.adoc#ous_change_log[`evergreen.ous_change_log()`] +
*Timing:* BEFORE INSERT OR UPDATE ROW

[CAUTION]
====
This trigger modifies the row before it is written (returns a modified `NEW`).
====

[source,plpgsql]
----

    DECLARE
    original TEXT;
    BEGIN
        -- Check for which setting is being updated, and log it.
        SELECT INTO original value FROM actor.org_unit_setting WHERE name = NEW.name AND org_unit = NEW.org_unit;
                
        INSERT INTO config.org_unit_setting_type_log (org,original_value,new_value,field_name) VALUES (NEW.org_unit, original, NEW.value, NEW.name);
        
        RETURN NEW;
    END;
----

==== log_ous_del

*Function:* xref:functions/evergreen.adoc#ous_delete_log[`evergreen.ous_delete_log()`] +
*Timing:* BEFORE DELETE ROW

[source,plpgsql]
----

    DECLARE
    original TEXT;
    BEGIN
        -- Check for which setting is being updated, and log it.
        SELECT INTO original value FROM actor.org_unit_setting WHERE name = OLD.name AND org_unit = OLD.org_unit;
                
        INSERT INTO config.org_unit_setting_type_log (org,original_value,new_value,field_name) VALUES (OLD.org_unit, original, 'null', OLD.name);
        
        RETURN OLD;
    END;
----
