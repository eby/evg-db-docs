= actor.hours_of_operation
:description: Table reference for actor.hours_of_operation

xref:schemas/actor.adoc[↑ actor schema]


When does this org_unit usually open and close?  (Variations
are expressed in the actor.org_unit_closed table.)


[WARNING]
====
*Cascading Deletes:* Deleting rows from this table will cascade to: xref:tables/actor/actor_org_unit.adoc[actor.org_unit].
====

[WARNING]
====
*Deferrable Constraints:* The following FK constraints are deferrable — they are checked at transaction end, not statement end: `hours_of_operation_id_fkey`.
====
