= acq.fund_allocation_total (view)
:description: View reference for acq.fund_allocation_total

xref:schemas/acq.adoc[↑ acq schema]

[NOTE]
====
This is a *database view*, not a base table. It has no triggers, indexes, or FK constraints of its own. Querying this view may be more efficient than joining the underlying tables directly.
====

== Columns

[cols="2,2,1,4"]
|===
|Column |Type |Nullable |Notes

| fund
| `integer`
| Yes
| 

| amount
| `numeric(100,2)`
| Yes
| 

|===

== View Definition

[source,sql]
----
 SELECT a.fund,
    sum(a.amount * acq.exchange_ratio(s.currency_type, f.currency_type))::numeric(100,2) AS amount
   FROM acq.fund_allocation a
     JOIN acq.fund f ON a.fund = f.id
     JOIN acq.funding_source s ON a.funding_source = s.id
  GROUP BY a.fund;
----
