= metabib.composite_attr_id_map (view)
:description: View reference for metabib.composite_attr_id_map

xref:schemas/metabib.adoc[↑ metabib schema]

[NOTE]
====
This is a *database view*, not a base table. It has no triggers, indexes, or FK constraints of its own. Querying this view may be more efficient than joining the underlying tables directly.
====

== Columns

| 

|===

== View Definition

[source,sql]
----
 SELECT c.id,
    c.ctype AS attr,
    c.code AS value
   FROM config.coded_value_map c
     JOIN config.record_attr_definition d ON d.name = c.ctype AND d.composite;
----
