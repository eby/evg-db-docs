= actor.stat_cat_entry_default
:description: Table reference for actor.stat_cat_entry_default

xref:schemas/actor.adoc[↑ actor schema]


User Statistical Category Default Entry

A library may choose one of the stat_cat entries to be the
default entry.


[WARNING]
====
*Cascading Deletes:* Deleting rows from this table will cascade to: xref:tables/actor/actor_org_unit.adoc[actor.org_unit], xref:tables/actor/actor_stat_cat_entry.adoc[actor.stat_cat_entry], xref:tables/actor/actor_stat_cat.adoc[actor.stat_cat].
====

[WARNING]
====
*Deferrable Constraints:* The following FK constraints are deferrable — they are checked at transaction end, not statement end: `stat_cat_entry_default_owner_fkey`, `stat_cat_entry_default_stat_cat_entry_fkey`, `stat_cat_entry_default_stat_cat_fkey`.
====
